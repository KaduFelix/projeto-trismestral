Aprendendo, criando e compartilhando!

Medida REM

A medida REM é uma unidade relativa que se baseia no tamanho da fonte do elemento raiz (normalmente o elemento <html>). Ela permite criar layouts flexíveis e escaláveis, facilitando a adaptação em diferentes tamanhos de tela.

Media Queries

As Media Queries permitem adaptar a aparência e o layout de um site para diferentes tamanhos de tela. Com o uso de regras CSS específicas, é possível aplicar estilos diferentes com base nas características do dispositivo, como largura da tela, orientação e resolução.

Responsividade

Um design responsivo é aquele que se adapta e se ajusta automaticamente em diferentes dispositivos e tamanhos de tela. Isso é alcançado através do uso de técnicas como Media Queries, Grids Flexíveis e ajustes de layout para garantir uma experiência de usuário consistente em qualquer dispositivo.

GitHub e Vercel

O GitHub é uma plataforma de hospedagem de código que permite versionar, colaborar e compartilhar projetos. A Vercel, por sua vez, é uma plataforma de deploy que facilita a publicação e o gerenciamento de sites estáticos, oferecendo integração com o GitHub.

Flexbox

O Flexbox é um modelo de layout em CSS que facilita a criação de designs flexíveis e responsivos. Com o uso de propriedades como display: flex, flex-direction e justify-content, é possível organizar elementos em uma única linha ou coluna, distribuí-los e alinhá-los de forma flexível.

Variáveis CSS

As variáveis CSS são valores nomeados que podem ser reutilizados em todo o seu código. Elas facilitam a manutenção e a personalização do estilo, permitindo que você defina valores em um único local e os utilize em várias propriedades. Isso torna mais fácil ajustar e atualizar o visual do seu site.
